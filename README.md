# wheatRust
Generate a favourable condicion map for the wheat rust to expand


## Data structure
Long,Lat,Graupel,Hail,Rain,Tmax,Tmin,Tpro,Dpoint,Hr,Windpro,WindDir, Hrmin, Hrmax, TprSoil0_10,TprSoil10_40, WprSoil0_10,WprSoil10_40

## Model
25 < Tpro < 30<br>
15 < Tmax - Tmin < 20<br>
Dpoint > 5<br>
